    Printing Labels for LumenPnP Machines

    1. Navigate to https://opulo.aligni.com/builds/and select the LumenPnP build that's due next

    2. Click "BLD-100### • LumenPnP"

    3. Press [cmd] + [c] to copy the Build Number

    Create LumenPnP V4 Outer Box Serial Labels

    1. Navigate to to tab My Account - Projects | Avery Print

      • For the login credentials go to the Manufacturing Account Login Info page linked here in Mattermost

    2. Select the "LumenPnP V4.X.X - Box Sticker" project

    3. Click "Open Project"

    4. Click the currently listed Build Number (100847-01 in this example)

    5. Click "SequentialNumbers"

    6. Click the "Prefix:" field

    7. Select the six digit build number used in the previous build

    8. Press [cmd] + [v] to paste in the new Build Number

    9. Double-click the "End Value:" field.

    10. Type "8" (or however many units this build is for)

    11. Click "Edit Sequential Numbers" to save your changes

    12. Click outside of the Sequential Numbers dialogue window

    13. Click "Preview & Print"

    14. Click "Print"

    15. Configure the print settings:

    16. Destination: Dymo LabelWriter 4XL
    17. Pages: 1-8
    18. Paper size: 104 x 159mm (4" x 6")

    19. Scale: Fit to printable area

      After the above settings have been applied go ahead and click the "Print" button

    20. The LumenPnP V4 Outer Box Serial Labels will now print out - set these aside for use on the production line!

    21. Click "Save Project"

    22. Click "Save to Your Account"

    23. Click "Got It"

    Create Serialized Machine ID Labels for the Control Boxes

    1. Navigate to theMachine ID Serial Label - Google Slides webpage

    2. Press [cmd] + [f] to open the "Find in document" pop-up window

    3. Click to expand the "Find" options before choosing "Find and Replace..."

    4. Type the previously used Build Number in the "Find" text field
      (which would be "100847" in this example)

    5. Click the "Replace with" text field.

    6. Press [cmd] + [v] to paste in the Build Number copied from Aligni

    7. Click "Replace all"

    8. Copy the expected finalization date (MFD) from the in-progress LumenPnP build

      • Note that if this date semems wrong please instead use the expected nearest Friday that the current LumenPnP build should be completed by

    9. Repeat the "Find and Replace" process to change the MFD from the previous Build completion date to the current Build completion date

    10. Exit the "Find and Replace" pop-up window

    11. Click "Print"

    12. Print the requisite Machine ID Labels

      • Set the Destination to "Dymo LabelWriter 4XL"

        • Note - you may need to add this paper size to your computer as a custom size and this is a pain in the butt so ask for help here!

      • Set the Scale to "Fit to printable area"

      • If you are printing labels for 8 machines, print pages 2-9
        • The first page of the Google Slide Show is just a "How to Edit" instructional page

      • Once you are ready, click "Print" to make the labels!
